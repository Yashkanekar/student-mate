### Tech stack used

- React.js
- Javascript
- CSS Modules
- Redux-toolkit

### Setup instructions

1. Get Open AI API Key
2. Add your API key in server/.env
3. Clone the repo
   ```sh
   git clone https://github.com/Yashkanekar/student-mate.git
   ```
4. Install NPM packages

   ```sh
   cd client/
   npm install

   cd server/
   npm install

   ```

5. Run the project from root directory:
   ```sh
   npm run dev
   ```
